
# ServiceNow UI Policy Project

## Project Overview
This project implements a UI Policy on the Incident table to enforce data quality 
by making the Priority field mandatory when an incident moves to the In Progress state.

## Features
- UI Policy: Priority field mandatory on In Progress state
- UI Action: State transition guidance for users
- Update Set: Export → Import → Commit migration flow
